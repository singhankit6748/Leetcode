# Roman to Integer

Roman numerals are represented by seven different symbols:

| Symbol | Value |
|--------|-------|
| I      | 1     |
| V      | 5     |
| X      | 10    |
| L      | 50    |
| C      | 100   |
| D      | 500   |
| M      | 1000  |

### Examples:
- **III** = 3
- **LVIII** = 58 (`L=50`, `V=5`, `III=3`)
- **MCMXCIV** = 1994 (`M=1000`, `CM=900`, `XC=90`, `IV=4`)

### Constraints:
- 1 <= s.length <= 15
- Characters: I, V, X, L, C, D, M
- Valid values: 1 to 3999
